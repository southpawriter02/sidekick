## File Entry Metadata

Each entry tracks:
- **Path:** Absolute file path
- **Name/Extension:** File name and type
- **Project:** Associated project
- **Open Count:** Access frequency
- **Last Opened:** Timestamp with date category
- **Pinned:** Whether pinned to top
- **Language/Icon:** Based on extension
