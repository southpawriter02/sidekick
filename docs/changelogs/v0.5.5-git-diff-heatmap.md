### ðŸ”¥ Git Diff Heatmap

- **Gutter Heatmap:** Color-coded indicators in the editor gutter
- **4 Metrics:**
  - **Commit Count:** Frequency of changes (Hotspots)
  - **Recency:** Time since last change
  - **Author Count:** Number of contributors
  - **Churn Rate:** Changes over time
- **3 Color Schemes:** Fire, Plasma, Viridis
- **Tooltips:** Detailed line stats (Last author, Last changed date, Commit hash)
- **Git Integration:** Direct integration with Git4Idea for reliable blame data
