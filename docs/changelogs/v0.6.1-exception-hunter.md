## Summary

This release adds **Exception Hunter** ‚Äî proactive detection of potentially unhandled exceptions with IDE inspection and quick fixes.

---

## New Features

### üîç Exception Hunter

- **Exception Detection:** Analyzes code for unhandled exceptions
- **5 Severity Levels:**
  - **Critical:** NullPointer, OutOfMemory, StackOverflow
  - **High:** IO, SQL, Security, Network
  - **Medium:** Illegal*, Unsupported*
  - **Low:** Other exceptions
  - **Info:** Documented/expected
- **Pattern Detection:** Identifies throwing calls (`.read()`, `.execute()`, `!!`, etc.)
- **IDE Inspection:** Integrated with IntelliJ inspection framework
- **Quick Fixes:**
  - Wrap in try-catch
  - Add throws declaration
  - Suppress warning
- **Configurable:** Min severity, ignored exceptions, gutter/highlight options

## Components Added

### Models (`ExceptionHunterModels.kt`)

| Type | Description |
|------|-------------|
| `UnhandledException` | Detected exception with type, location, severity |
| `ExceptionLocation` | File, line, column, method/class context |
| `CallSite` | Call chain representation |
| `ExceptionSeverity` | 5 levels with type-based classification |
| `ExceptionHunterConfig` | Feature configuration with presets |
| `ExceptionAnalysisResult` | Analysis output with metrics |

### Service (`ExceptionHunterService.kt`)

| Feature | Description |
|---------|-------------|
| PSI Analysis | Traverses PSI tree for exception detection |
| Pattern Matching | Detects throw statements and throwing APIs |
| Caching | In-memory cache of analysis results |
| Handled Detection | Checks for try-catch and throws declarations |

### Inspection (`ExceptionHunterInspection.kt`)

| Component | Description |
|-----------|-------------|
| `ExceptionHunterInspection` | LocalInspectionTool implementation |
| `AddTryCatchFix` | Quick fix to wrap in try-catch |
| `AddThrowsDeclarationFix` | Quick fix to add throws clause |
| `SuppressExceptionFix` | Quick fix to ignore exception type |
