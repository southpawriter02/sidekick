## Summary

This release adds **TODO Tracker** ‚Äî deadline-aware TODO management with IDE tool window.

---

## New Features

### üìù TODO Tracker

- **Comment Scanning:** Extracts TODOs from code comments via PSI
- **8 TODO Types:**
  - TODO, FIXME, HACK, BUG, NOTE, OPTIMIZE, REVIEW, DEPRECATED
- **4 Priority Levels:**
  - Critical (!!!), High (!!), Medium, Low
- **Deadline Detection:**
  - ISO dates (2024-12-31)
  - US format (by 12/31/2024)
  - Due prefix (due: date)
- **Author & Tags:** @mentions and #hashtags
- **Status Tracking:** Open, Due Soon, Overdue, Completed
- **Tool Window UI:**
  - Table view with all project TODOs
  - Filtering by type, status, priority
  - Click-to-navigate to source
  - Color-coded urgency indicators
