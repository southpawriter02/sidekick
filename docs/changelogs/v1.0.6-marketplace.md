# v1.0.6 - Marketplace & Documentation

> **Release Date:** 2026-02-06  
> **Category:** Distribution

---

## Overview

This version provides **Marketplace & Documentation** assets for JetBrains Marketplace submission.

---

## New Features

### Marketplace Module (`com.sidekick.marketplace`)

#### MarketplaceAssets.kt

| Type | Purpose |
|------|---------|
| `MarketplaceMetadata` | Plugin ID, name, version, description, tags |
| `DocumentationSite` | Documentation structure with sections/pages |
| `DocSection` / `DocPage` | Section and page definitions |
| `ScreenshotAsset` | Marketing screenshot metadata |
| `MarketplaceSubmission` | Submission checklist with 10 items |
| `Tutorial` | Getting started tutorial structure |

---

## Submission Checklist

```
[ ] Plugin builds without errors
[ ] All tests pass
[ ] Plugin.xml validates
[ ] Screenshots prepared (min 3)
[ ] Description formatted (HTML)
[ ] Change notes complete
[ ] Icon created (40x40 SVG)
[ ] Signed with certificate
[ ] README updated
[ ] License file included
```

## Documentation Structure

- **Getting Started** — Installation, Configuration, First Chat
- **Features** — AI Chat, Code Actions, Coding Agent
- **Providers** — Ollama, LM Studio, Cloud Providers
- **API Reference** — Extensions, Tools, Templates

## Usage Example

```kotlin
// Generate plugin.xml content
val metadata = MarketplaceMetadata(version = "1.0.0")
val xml = metadata.toPluginXml()

// Create submission checklist
val submission = MarketplaceSubmission()
val updated = submission
    .complete("Plugin builds without errors")
    .complete("All tests pass")

println("Progress: ${updated.completionPercent}%")
```
