# v1.1.1 — Chat Export to Markdown

**Released:** 2026-02-07

## Summary

Export chat conversations as well-formatted Markdown files, or copy individual
messages to the clipboard in Markdown format.

## New Features

### Per-Bubble "Copy as Markdown" Button

- Each message bubble now displays a small clipboard icon after streaming
  completes
- Click the icon to copy that single message as Markdown to your clipboard
- User messages get a `**You:**` prefix; assistant messages get a
  `**Sidekick:**` prefix; errors render as `> ⚠️` block-quotes

### Full Conversation Export

- New **Export** button in the Sidekick chat toolbar (header panel)
- Opens a file-save dialog to write the entire conversation as a `.md` file
- Includes metadata header: export timestamp, project name, and model name
- Also available via the action menu: **Sidekick: Export Chat as Markdown**
